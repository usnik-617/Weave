# 위브(Weave) UI 문서

## 1. 브랜드/톤
- 주요 브랜드 컬러: 하늘색 계열(상단 네비게이션/푸터 기준)
- 가독성 우선의 밝은 테마 유지

## 2. 공통 UI 원칙
- 데스크톱/모바일 정보 밀도 분리
- 리스트 툴바: 좌측 통계, 우측 정렬/검색/글쓰기 정렬 유지
- 버튼 텍스트는 줄바꿈 없이 표시(text-nowrap)

## 3. 주요 화면 패턴
- 목록: 검색 + 정렬 + 페이지네이션
- 상세: 메타정보 + 본문 + 댓글 + 이전/다음 네비
- 작성: Rich Editor + 이미지 업로드 큐/진행 상태

## 4. 명예의 전당 UI
- 금/은/동 그룹 섹션 구조
- 카드형 이름 노출 + 관리자 수정/삭제 액션
- 검색/페이지네이션/감사로그(권한 기반)

## 5. 모바일 정책
- 하단 탭은 한 줄 유지
- 오프캔버스 메뉴는 현재 상위/하위 탭만 강조
- 탭 전환 체감 개선(애니메이션 및 상태 복원)

## 6. 접근성/품질
- 폼 입력 라벨 연결
- 상태/오류 메시지의 명확성
- 텍스트/버튼 대비 유지
