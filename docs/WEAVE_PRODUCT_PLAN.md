# 위브(Weave) 홈페이지 기획서 (운영 초안)

## 1. 서비스 목적
- 남양주자원봉사센터 소속 위브 봉사단의 공지, 활동, 참여를 PC/Mobile 반응형으로 통합 제공
- 운영진/관리자 중심의 게시 관리 + 일반/단원 중심의 참여 흐름 제공

## 2. 사용자 역할
- GENERAL(일반): Q&A 작성/조회 중심
- MEMBER(단원): Q&A 작성/조회 + 봉사 참여 흐름
- EXECUTIVE/LEADER/VICE_LEADER/ADMIN: 공지/갤러리/운영 기능 관리

## 3. 핵심 기능 범위
- 홈: 주요공지/요약 통계/캘린더 미리보기
- 소개: 연혁/임원진/로고/규정
- 활동: 활동소개/캘린더/일정 상세
- 갤러리: 연도 탭, 카드형 목록, 상세, 이미지 다중 업로드
- 소식: 공지/FAQ/Q&A, 상세, 검색/정렬
- 참여: 명예의 전당/문의/후원
- 인증: 회원가입/로그인/아이디·비밀번호 찾기

## 4. 권한 정책(요약)
- 일반/단원: 글쓰기에서 Q&A만 허용
- 운영진 이상: 공지/갤러리 작성 및 관리 기능 허용
- 관리자/운영진: 명예의 전당 감사로그 노출

## 5. 비기능 요구
- 반응형 UI(모바일 우선 사용성 포함)
- 100+ 동시접속 대비(캐시/큐/오브젝트스토리지/DB 확장)
- 이미지 정책: 최대 30장, 총량 300MB, 변환/압축/썸네일 전략

## 6. 향후 확장
- PostgreSQL 전환 가속
- RQ/Celery 백그라운드 작업 분리
- CDN/오브젝트 스토리지 본격 운영
